# 0.2.0 (21 Jun 2019)

## BREAKING CHANGES

* `wait.for(n).milliseconds()` simplified to `wait.milliseconds(n)`
* `wait.for(n).seconds()` simplified to `wait.seconds(n)`

## Added Features

* `wait.condition` added; can now wait for a condition to become true
